# class Knife

## Attributes:

* isSharp (boolean)
* bladeLength (int)
* objectsCut (array)
* steelType (string)

## Methods:

* sharpenKnife (changes isSharp to true)
* breakKnife (subtracts 1 from bladeLength)
* cutProduce (adds elements cut to objectsCut array)
* cutRocks (changes isSharp to false)
